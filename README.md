# Cert_Tester
Dieses interaktive Tool testet den Win 11 Computerzertifikats und Benutzerzertifikatsstore auf die Echtheit der Zertifiktate, erstellt ein Backup und löscht Zertifikate die Untrusted sind. Das Backup kann man zurückspielen und noch einen HTML-Report über alle Zertifikate erstellen.

.\Cert_Prüfung.ps1  startet das interaktive Programm, zur Auswahl stehen 6 Unterrubriken, die man aufrufen kann
